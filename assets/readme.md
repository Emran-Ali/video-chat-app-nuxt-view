# Assets Directory

The `assets` directory is used to store various static assets for the Pengine Admin Dashboard. These assets include:

- **Images**: Icons, logos, and other image files used throughout the application.
- **Stylesheets**: CSS files that define the styling of the application.
- **JavaScript**: JavaScript files that provide additional functionality to the application.
- **Fonts**: Custom fonts used in the application.
- **Data**: JSON or other data files used for populating the application with content.

## Directory Structure

```
assets/
├── images/
├── css/
├── javascript/
├── fonts/
└── data/
```

Each subdirectory is organized to keep the assets manageable and easy to locate.

## Usage

- **Images**: Place all image files in the `images` directory.
- **Stylesheets**: Place all CSS files in the `stylesheets` directory.
- **JavaScript**: Place all JavaScript files in the `javascript` directory.
- **Fonts**: Place all font files in the `fonts` directory.
- **Data**: Place all data files in the `data` directory.

Ensure that all assets are optimized for performance and follow the naming conventions used throughout the project.
